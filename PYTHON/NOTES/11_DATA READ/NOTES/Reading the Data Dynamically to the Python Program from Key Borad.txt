				===============================================================
					Reading the Data Dynamically to the Python Program from Key Board
				===============================================================
=>In Python Programming, to read the Data Dynamically from Key Board, we have Two Pre-Defined Functions. They are

			1. input()
			2. input(Message)
-----------------------------------------------------------------------------------------------------------------------------------------------------------
1. input()
-----------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:     varname=input()
=>This function is used for Reading any type of data from Key Board in the form of str and placed in LHS Variable 
    Name and whose type is <class, 'str'>
=>To get str data in our own value type, we must use type casting functions.
-----------------------------------------------------------------------------------------------------------------------------------------------------------
2. input(Message)
-----------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:  varname=input(Message)
=>here Message is one of String and It Represents User-Prompting Message
=>This function is used for Reading any type of data from Key Board in the form of str in LHS Variable 
    Name and whose type is <class, 'str'> and additionally we can also Give User-Prompting Message.
-----------------------------------------------------------------------------------------------------------------------------------------------------------
NOTE:   print() is called OUTPUT Statement
	      input() and input(message) is called INPUT Statements
