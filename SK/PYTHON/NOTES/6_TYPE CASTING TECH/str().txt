				==============================================
							5. str()
				==============================================
=>str() is used for Coverting All types of Values into str.
=>Syntax:     varname=str(int/float/bool/complex)
-------------------------------------------------------------------------------------------------------------------------------------------------------------
Examples
-------------------------------------------------------------------------------------------------------------------------------------------------------------
>>> a=12
>>> print(a,type(a))--------------12 <class 'int'>
>>> s=str(a)
>>> print(s,type(s))--------------12 <class 'str'>
>>> s---------------------------------'12'
---------------------------
>>> a=3.12
>>> print(a,type(a))---------------3.12 <class 'float'>
>>> s=str(a)
>>> print(s,type(s))--------------3.12 <class 'str'>
>>> s----------------------------------'3.12'
------------------------------
>>> a=True
>>> print(a,type(a))--------------True <class 'bool'>
>>> s=str(a)
>>> print(s,type(s))--------------True <class 'str'>
>>> s--------------------------------'True'
---------------------------------
>>> a=2+3j
>>> print(a,type(a))------------(2+3j) <class 'complex'>
>>> s=str(a)
>>> print(s,type(s))------------(2+3j) <class 'str'>
-------------------------------------------------------------------------------------------------------------------------------------------------------------
