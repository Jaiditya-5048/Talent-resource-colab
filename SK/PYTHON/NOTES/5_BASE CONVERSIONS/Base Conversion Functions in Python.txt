		==============================================================
					Base Conversion Functions in Python
		==============================================================
=>In Python Programming, we have 3 Base Conversion Functions. they are

				1. bin()
				2. oct()
				3. hex()
----------------------------------------------------------------------------------------------------------------------------------------------------------
1. bin()
----------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:  varname=bin(Decimal OR Octal OR Hexa Decimal Value) 
=>This Function is used for Converting Decimal, Octal and Hexa Decimal value into Binary value.
--------------------
Examples
--------------------
>>> a=12
>>> b=bin(a)
>>> print(b)--------------------0b1100
----------------------
>>> a=0o12
>>> b=bin(a)
>>> print(b)--------------------0b1010
-----------------------
>>> a=0xD
>>> b=bin(a)
>>> print(b)-------------------0b1101
----------------------------------------------------------------------------------------------------------------------------------------------------------
2. oct()
----------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:  varname=oct(Decimal OR Binary OR Hexa Decimal Value) 
=>This Function is used for Converting Decimal,Binary and Hexa Decimal value into Octal value.
-----------------
Examples
------------------
>>> a=12
>>> ov=oct(a)
>>> print(ov)--------------0o14
----------------------
>>> a=0b1010
>>> ov=oct(a)
>>> print(ov)-------------0o12
----------------------
>>> a=0xF
>>> ov=oct(a)
>>> print(ov)-------------0o17
----------------------------------------------------------------------------------------------------------------------------------------------------------
3. hex()
----------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax:   varname=hex(Decimal OR Binary OR Octal )
=>This Function is used for Converting Decimal, Binary and Octal Values into Hexa Decimal values.
----------------------------
Examples
----------------------------
>>> a=15
>>> hv=hex(a)
>>> print(hv)-------------------0xf
-------------------------
>>> a=0b1010
>>> hv=hex(a)
>>> print(hv)------------------0xa
---------------------------
>>> a=0o17
>>> hv=hex(a)
>>> print(hv)------------------0xf
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
