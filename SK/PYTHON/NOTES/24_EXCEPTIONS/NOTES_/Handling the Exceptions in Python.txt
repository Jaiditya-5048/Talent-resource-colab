			==================================================
				Handling the Exceptions in Python
			==================================================
=>Handling the Exceptions is nothing Converting Technical Error Messages into User-Friendly Error Messages.
=>For Converting Technical Error Messages into User-Friendly Error Messages, we have 5 Key words in Exception Handling Concept. They are
			
					1. try
					2. except
					3. else
					4. finally
					5. raise
----------------------------------------------------------------------------------------------------------------------------------------------------------
Syntax for Handling the Exceptions
----------------------------------------------------------------------------------------------------------------------------------------------------------
		try:
		     Block of Statements
		     Generates Exceptions
		except exception-class-name-1:
			Block of Statements
			generates User-Friendly Error Messages
		except exception-class-name-2:
			Block of Statements
			generates User-Friendly Error Messages
		---------------------------------------------
		---------------------------------------------
		except exception-class-name-n:
			Block of Statements
			generates User-Friendly Error Messages
		else:
			Block of Statements
			generates results of the programs
		finally:
			Block of Statements
			Executes Compulsorily
